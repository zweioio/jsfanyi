简单几步配置，即可在浏览器中享受最精准的翻译体验

01

下载并解压

点击上方下载按钮获取插件包，并解压到您容易找到的本地文件夹中。

02

开启开发者模式

打开 Chrome 浏览器，进入 chrome://extensions/ 并开启右上角的“开发者模式”。

点击“加载已解压的扩展程序”，选择解压后的文件夹。点击侧边栏图标即可开始翻译。

常见问题

插件支持哪些浏览器？

目前深度适配 Google Chrome 和 Microsoft Edge 浏览器，理论上支持所有基于 Chromium 内核的浏览器。

如何配置 AI 智能引擎？

在插件设置页面，您可以填入自己的 OpenAI 或 Google Gemini API Key 即可启用 AI 智能引擎。

翻译数据会上传吗？

不会。我们非常重视您的隐私，所有翻译请求均直接发送至对应的翻译引擎 API，插件本身不存储任何内容。
